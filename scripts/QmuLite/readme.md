# QMULITE

Script para ejecutar maquinas virtuales con QEMU-KVM.


## Instalacion:

Ejecutar `./instalar` (pedira contraseña)

Ya podras ejecutar QMULITE en tu terminal con `qmulite`


## Desinstalar:

Ejecutar `./desinstalar` (pedira contraseña)
